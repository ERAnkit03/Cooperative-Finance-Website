# 📖 About The Project

The **Cooperative Finance Website** is a professional frontend web application developed to provide an intuitive digital experience for customers of a cooperative financial institution.

The application emphasizes **performance**, **responsiveness**, **clean UI**, and **maintainable architecture**, making it suitable for real-world deployment.

> **Privacy Notice**
>
> This repository is a sanitized portfolio version. All client-specific branding, logos, contact details, and confidential assets have been removed.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/cooperative-finance-website.git
```

## Navigate

```bash
cd cooperative-finance-website
```

## Install Packages

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

---

# 📦 Production Build
